How to get a free Groq API key

1
Go to console.groq.com

Open your browser and navigate to console.groq.com

2
Create a free account

Sign up with Google or email — no credit card needed

3
Go to "API Keys" in the sidebar

Click on API Keys in the left navigation panel

4
Create a new API key

Click "Create API Key" and give it any name

5
Copy and paste it here

The key starts with gsk_ — paste it into the API key field above

Groq is free and uses Meta's Llama model to generate your cards. Your key is stored locally in your browser — it never leaves your device.
